# Länderbeispiele — typische Tendenzen

Aus dem HWZ-Skript (Uehlinger) und ergänzenden Quellen (Hofstede, Meyer). **Tendenzen, nicht Wahrheiten.** Person, Branche, Generation, Ausbildung im Ausland überschreiben das oft.

## Schweiz (DACH-Standard mit Schweizer Eigenarten)
Universalismus / Individualismus mittel / Spezifisch (eher pfirsich-nah, mit Schweizer Reserve) / Status erreicht / Neutral / Monochron / Niedrige Machtdistanz / Niedriger Kontext (aber höher als DE) / Linear / Eher ideenorientiert / Abstrakt / Abgetrennt.

Schweizer Eigenart: indirekter als deutsche, höflicher in der Form, distanzierter beim Du. Konsensorientiert.

## Deutschland
Wie CH, aber: niedrigerer Kontext (sehr direkt), abstrakter, distanzierter. Punkt-für-Punkt-Argumentation. "Wenn ich nichts sage, ist es ok" gilt nicht — explizit erwarten.

## Frankreich
Eher partikularistisch / Individualismus / Diffus / Status zugeschrieben (Bildungsstatus, Grandes Écoles) / Eher affektiv / Eher polychron / Höhere Machtdistanz als DACH / Höherer Kontext / Eher zirkulär / Beziehungsorientiert in Disagreement / Eher abstrakt / Eher verbunden.

## Niederlande
Stark universalistisch / sehr individualistisch / spezifisch / Status erreicht / neutral / monochron / **sehr niedrige Machtdistanz** (extrem flach) / **sehr niedriger Kontext** / linear / **stark ideenorientiert** (direkte Kritik ist Normalkommunikation) / mittel / abgetrennt.

## Italien / Spanien
Eher partikularistisch / Eher kollektivistisch (Familie zentral) / Diffus / Status teils zugeschrieben / **Affektiv** / Polychron / Mittlere Machtdistanz / Mittlerer bis höherer Kontext / Eher zirkulär / Beziehungsorientiert / Konkret / **Verbunden**.

## USA
Universalismus / **Stark individualistisch** / Pfirsich (sehr offen, oberflächlich nahe) / Status erreicht / Mittel / Monochron / Niedrige Machtdistanz / Niedriger Kontext / Linear / Eher ideenorientiert / Konkret / Mittel verbunden.

US-Eigenart: Optimismus ist Pflicht. "Awesome!" ist nicht Lob, sondern Standard.

## Japan
Universalismus mit Beziehungspflege / **Kollektivistisch** / **Kokosnuss** (sehr lange Annäherung) / Status zugeschrieben (Alter, Senioritätsrang) / **Sehr neutral** (Gefühle versteckt) / Monochron / Hohe Machtdistanz / **Sehr hoher Kontext** / Zirkulär / Stark beziehungsorientiert / Konkret / Abgetrennt.

JP-Eigenart: "Nein" wird fast nie direkt gesagt. "Es ist schwierig", "Wir prüfen es", "Vielleicht" sind Nein. Stille ist normal, Pausen sind Information.

## China
Universalismus mit starkem Guanxi (Beziehung) / Kollektivistisch (Familie + Gruppe) / Kokosnuss / Status zugeschrieben & erreicht / Mittel (kontextabhängig) / Mittel polychron / Hohe Machtdistanz / Hoher Kontext / Eher zirkulär / Beziehungsorientiert / Konkret / Mittel.

CN-Eigenart: Mianzi (Gesicht) zentral. Direkte Kritik in Gruppe = massiver Affront.

## Indien
Partikularismus / Kollektivistisch / Diffus / Status zugeschrieben (Familie, Bildung, Kaste — auch wenn offiziell nicht thematisiert) / Mittel affektiv / **Polychron** / Hohe Machtdistanz / Höherer Kontext / Zirkulär / Beziehungsorientiert / Konkret / Verbunden.

IN-Eigenart: "Yes" heisst oft "I hear you", nicht "I agree". Pünktlichkeit ist nachlässig — Beziehung schlägt Termin.

## Brasilien / Mexiko / LATAM-Schnitt
Partikularistisch / Kollektivistisch / Diffus / Status zugeschrieben / Sehr affektiv / Polychron / Hohe Machtdistanz / Höherer Kontext / Zirkulär / Sehr beziehungsorientiert / Konkret / Sehr verbunden.

## Saudi-Arabien / VAE / arabische Welt
Partikularistisch / Kollektivistisch (Familie + Stamm) / Diffus / Status zugeschrieben (Familie, Religion, Alter) / Affektiv (kontrolliert in der Geschäftswelt) / Polychron / Sehr hohe Machtdistanz / **Sehr hoher Kontext** / Zirkulär / Sehr beziehungsorientiert / Konkret / Mittel.

MENA-Eigenart: Vor Geschäft sehr lange Beziehungspflege. Erste 2–3 Treffen oft kein Inhalt — Kennenlernen, Vertrauen.

## Russland
Partikularismus / Kollektivismus mit starkem Individualismus in Führung / Kokosnuss / Status zugeschrieben / Neutral in Geschäft, affektiv privat / Polychron / Hohe Machtdistanz / Hoher Kontext / Eher zirkulär / Beziehungsorientiert / Mittel / Mittel.

## Schweden / Dänemark / Norwegen
Universalismus / Individualismus / Spezifisch / Status erreicht / Neutral / Monochron / **Niedrigste Machtdistanz weltweit** / Niedriger Kontext / Linear / Ideenorientiert / Mittel / Abgetrennt.

Sehr konsensorientiert. Entscheidungen lange dauern — alle müssen einbezogen werden.

---

## Reminder

Diese Liste ist eine Grobeinschätzung auf Länderebene. **Individuelle Personen sind oft anders.** Mehr-Kultur-Personen (im Ausland aufgewachsen, internationale Karriere) haben in der Regel Mischprofile. Bei wichtigen Beziehungen: lieber beobachten und fragen, statt Stereotypen anwenden.
