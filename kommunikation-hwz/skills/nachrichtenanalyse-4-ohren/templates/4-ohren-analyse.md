## Reaktionsoptionen

| Option | Beschreibung | Trade-off |
|---|---|---|
| 1. Sachebene | Nur auf Fakten antworten | Beziehungssignal bleibt unbehandelt |
| 2. Beziehung explizit | "Habe ich das richtig…?" | Kostet Energie, setzt Vertrauen voraus |
| 3. Nachfragen | "Was meinst du genau mit Y?" | Holt Info, ohne sich festzulegen |
| 4. Verzögern | Schlaf drüber, morgen antworten | Brückenzeit, dafür wirkt es evtl. ignorant |
| 5. Persönlich | Telefon / Treffen vorschlagen | Bestes Format für Beziehungsfragen — aber Aufwand |
