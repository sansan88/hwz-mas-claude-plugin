# Redenvorbereitung — Vorlage

## 1. Grundlagen

- **Anlass:** ___
- **Redeart:** Sachvortrag / Überzeugungsrede / Gesellschaftsrede / Stegreif / Pitch / Statement
- **Zeit:** ___ Minuten
- **Publikum:** ___ (Anzahl, Hintergrund, Vorwissen, Erwartung, Stimmung)
- **Ort / Setting:** ___ (Bühne, Raum, Stuhl/stehend, mit/ohne Slides)
- **Redeziel — in einem Satz:** *Am Ende sollen die Zuhörer ___ wissen / fühlen / tun.*

## 2. Inventio — Stoff sammeln

Brainstorm, mehr als du brauchst. Argumente, Beispiele, Geschichten, Zahlen, Zitate, Vergleiche, Bilder.

## 4. Elocutio — sprachliche Gestaltung

Konkret statt abstrakt. Bilder statt Begriffe. Aktiv statt Passiv. Kurz statt lang.

- Wo benutze ich Floskeln, die ich streichen kann? ___
- Wo brauche ich ein Bild / Beispiel? ___
- Gibt es einen wiederkehrenden Begriff / Motiv? ___
