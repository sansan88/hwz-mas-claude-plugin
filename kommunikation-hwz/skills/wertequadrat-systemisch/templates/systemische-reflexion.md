# Systemische Reflexion — Vorlage

Für Situationen, in denen du selbst feststeckst oder über jemand anderen / über ein Team / über eine Beziehung nachdenkst.

## 1. Beobachtung

Was beobachte ich? Ohne Bewertung, nur Verhalten.

## 2. Mein erstes Urteil

___

## 4. Zirkuläre Frage

Wie würde diese Person ihr eigenes Verhalten beschreiben?
___

Was glaubt sie wahrscheinlich, was ich gerade von ihr denke?
___

Wer in ihrem Umfeld profitiert davon, dass sie sich so verhält?
___

## 5. Reframing

Was wäre, wenn ich das Verhalten anders benenne — als Stärke statt als Schwäche?
___

Welche Funktion erfüllt das Verhalten im System (auch wenn ich sie nicht teile)?
___
